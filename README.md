# Zen Type

A minimalist [Monkeytype](https://monkeytype.com/) clone focused on the Portuguese language. Built with AI.

**[Access the project here](https://zen-type-tau.vercel.app/)**

## How to run

1. Install [Node.js](https://nodejs.org/).

2. Clone the repository:
```bash
git clone https://github.com/phbrg/zen-type.git
```

3. Open the project on your favorite IDE and run the command:
```
npm run dev
```
